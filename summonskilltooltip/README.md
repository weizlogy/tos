### v1.0.0

#### 新規作成

アドオンがロードされると、召喚獣にオベイした際に使えるクイックスロットのスキル（降りるを除く）にツールチップを表示できるようにします。

通常のスキル同様、マウスホバーで表示されます。

ツールチップでは、該当スキルのクラスタイプ、攻撃タイプ、スキル威力、クールダウンタイムを表示します。
